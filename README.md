# 11ty-Serene

A personal page using [11ty-Serene](https://github.com/CLCK0622/11ty-Serene) theme.

Made by [CLCK](https://github.com/CLCK0622).

## Install (all scripts are using `pnpm` by default)

```
pnpm install
```

## Build

```
pnpm run build
```

## Start

```
pnpm run watch:css
pnpm run start
```
